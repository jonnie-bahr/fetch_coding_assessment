# Fetch Coding Assessment: Points
Coding assessment for backend software engineer position at Fetch Rewards

# Getting Started

## Prerequisities
- Python3 installed

## Install packages in shell
```
bash
cd fetch_coding_assessment
make setup
```

## Run API in shell
```
bash
cd fetch_coding_assessment
make run
```

## Run tests in shell
```
bash
cd fetch_coding_assessment
make test
```
